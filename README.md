### 💡 Overview
Designed and developed a **multi-agent AI system** to intelligently discover, plan, and monitor AWS cost optimization opportunities.

- 🤖 Multi-agent architecture: **Discoverer • Planner • Monitor • Assistant**
- 🧠 LLM-powered reasoning for **right-sizing and cost optimization strategies**
- 🔄 Human-in-the-loop workflows for **safe automation and decision validation**

---

### ⚙️ Architecture

- ⚡ **Base44 AI + n8n** for orchestration and agent workflows  
- ☁️ **AWS Lambda + EventBridge** for serverless processing  
- 🔐 **Cognito** for authentication and secure access  
- 📊 **React dashboard** for cost insights and visualization  

---

### 📊 Key Features

- 📉 Automated **cost optimization recommendations**
- 📈 Budget forecasting and usage trend analysis  
- 🔍 Intelligent resource discovery across AWS environments  
- 🧠 AI-driven decision-making with guardrails
